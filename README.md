# moy

`moy` 是基于模型框架 `kero` 和 `UI` 框架 `neoui` 实现的应用框架，快速构建数据驱动型应用。

## Introduction

`moy` — 取名自中文名墨渊的简称，寓意战神，强大，和值得信任的战斗力。

`moy` 核心思想是对 `MVVM` 架构中 `Model` 前端数据层的强化和沉淀，并通过独有的 `u-meta` 标示来实现从 `UI` 到数据的自动映射和数据绑定
